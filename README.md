
# Project Overview
Analysis of humanitarian funding data using SQL Server to identify funding trends, sector allocations, and country-level funding patterns.

# Questions Answered
- Funding trend over time
- Top funded sectors
- Lowest-funded countries
- Highest-funded countries
- Peak funding year
- Sector growth trends
- Country funding growth
# SQL Skills Demonstrated
- Aggregations
- Window Functions
- LAG()
- ROW_NUMBER()
- GROUP BY
- Ranking
- Percentage calculations
